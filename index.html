GitHub Heroku

Tech Stack

Ruby on Rails 5.0 | Javascript | AJAX | jQuery | HTML5 | Bootstrap | Heroku | Facebook OAuth 2 | Spoonacular API

Description

Forked is a social media web app for recipes. Built mainly in Ruby on Rails, Forked allows you to search for recipes based on what ingredients you have, cuisine, allergies, dish (main course, side dish), and a specific type of recipe.

Intramuralizer

GitHub Heroku

Tech Stack

Ruby on Rails | Twilio | Google Maps API | HTML | CSS | Bootstrap

Description

A Rails web application created in 2 days. It is designed to be a convenient way for coaches and players in intramural sports teams to view leagues,teams, games, and win/loss Records. New coaches may create and update teams, add players, and create a date and time location.

Tech Stack
